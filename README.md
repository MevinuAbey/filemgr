# FileMgr : command-line file management utility
#### Video Demo:  <URL HERE>
#### Description:

FileMgr is a Python-based command-line file management utility .  

- **Organize files** into structured folders by file type, extension, type+extension, or modified date.
- **Rename files in bulk** using prefix, suffix, text replacement, or auto-numbering.
- **Create backups** with options for compressed ZIP backups, timestamped or overwrite modes, and file-type include/exclude filtering.
